Black bullet is an application that aims to provide android integration to Linux.

Currently supported functionalities:

  • Notifications integration
  • Battery Status
  • Remote ringtone playing
  • Clipboard shareing

Devices don't have to be in the same network to make it work. Simply download and install apk, then register in your phone and you are ready to go.

Install deb package on your linux or download sources from my github.
